# harshit.me

Static website for harshit.me.

## Commands

Run hugo server.

```bash
hugo server
```

Compile hugo site.

```bash
hugo
```

Deploy the site to play-mine server.

```bash
scp -r workspace/dev/untitled/harshit.me/blog/public/*  root@157.245.101.248:/var/www/harshit.me/
```

Path of the main website.

```bash
/var/www/harshit.me/
```
